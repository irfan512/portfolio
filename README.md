# 🚀 Modern Portfolio Website

A stunning, modern portfolio website built with React, Tailwind CSS, and GSAP animations. Features a single-page scrolling design with smooth animations and a beautiful user interface.

## ✨ Features

- **Modern Design**: Clean, professional design with gradient colors and glassmorphism effects
- **Single-Page Scrolling**: Smooth navigation between sections without page reloads
- **GSAP Animations**: Beautiful scroll-triggered animations and transitions
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Hover effects, smooth transitions, and engaging user interactions
- **Performance Optimized**: Fast loading and smooth scrolling experience

## 🛠️ Technologies Used

- **Frontend**: React 18
- **Styling**: Tailwind CSS
- **Animations**: GSAP (GreenSock Animation Platform)
- **Icons**: FontAwesome
- **Build Tool**: Create React App

## 🎯 Sections

1. **Hero Section**: Eye-catching introduction with animated elements
2. **About**: Personal information and skills overview
3. **Skills**: Technical skills with animated progress bars
4. **Experience**: Work history and education timeline
5. **Projects**: Portfolio showcase with filtering options
6. **Contact**: Contact form and information

## 🚀 Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

```
src/
├── Components/          # Reusable UI components
│   ├── Header.js       # Navigation header
│   ├── Hero.js         # Hero section
│   ├── About.js        # About section
│   ├── Skills.js       # Skills section
│   ├── Experience.js   # Experience timeline
│   ├── Projects.js     # Projects showcase
│   ├── Contact.js      # Contact form
│   └── Footer.js       # Footer section
├── assets/             # Images and static files
├── Details.js          # Personal information and data
├── App.js              # Main application component
├── index.js            # Application entry point
└── index.css           # Global styles and animations
```

## 🎨 Customization

### Personal Information

Edit `src/Details.js` to update:
- Personal details
- Work experience
- Education
- Projects
- Contact information
- Social media links

### Styling

- Modify `src/index.css` for custom CSS
- Update `tailwind.config.js` for Tailwind customization
- Change color schemes in component files

### Animations

- Adjust GSAP animations in component files
- Modify animation timing and effects
- Add new scroll-triggered animations

## 📱 Responsive Design

The portfolio is fully responsive and includes:
- Mobile-first design approach
- Breakpoint-specific layouts
- Touch-friendly interactions
- Optimized for all screen sizes

## 🚀 Deployment

### Build for Production

```bash
npm run build
```

### Deploy to Netlify

1. Push your code to GitHub
2. Connect your repository to Netlify
3. Deploy automatically on push

### Deploy to Vercel

1. Install Vercel CLI: `npm i -g vercel`
2. Run: `vercel`
3. Follow the prompts
